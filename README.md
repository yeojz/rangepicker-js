Rangepicker for jQuery UI
=======================

About
-----
Rangepicker is a wrapper/extension to provide for range selection within jQuery UI's datepicker.



Usage
---

**To initialise:**

	$("#elem").rangepicker();




**Options**

	$("#elem").rangepicker({
		datepicker: {
			numberOfMonths: [3,4],
	  	},
		periodToggle: [false, false]
	});
